# Instagram Growth Master

行動科学 x アルゴリズム理解に基づくInstagram成長スキル。
「なぜ伸びるのか」の心理学的根拠を持つ戦略で、再現性のある成長を実現する。

## コア原理

3つの成長エンジン（全て行動科学に裏打ち）:

1. **Attract（惹きつける）** - 非フォロワーへのリーチ拡大
   - リール・Explore経由の新規接触
   - パターンインタラプトでスクロール停止
   - 好奇心ギャップで「もっと見る」をタップさせる

2. **Convert（転換する）** - 閲覧者をフォロワーに変える
   - プロフィールの第一印象最適化（ハロー効果）
   - 固定投稿で価値を証明（アンカリング効果）
   - 「ここでしか得られない」希少性の設計

3. **Engage（深める）** - フォロワーをコミュニティに育てる
   - 返報性で双方向の関係構築
   - 習慣ループで定期的な閲覧を形成
   - 変動報酬で飽きさせない

## 2025-2026アルゴリズムの最重要ポイント

知っておくべき4つの変化:

| 変化 | 意味 | 対策 |
|------|------|------|
| **Sends（DM共有）が最重要指標に** | 「人に送りたい」コンテンツが最も伸びる | 実用的価値・社会的通貨の高いコンテンツ |
| **小規模クリエイター優遇** | フォロワー数<コンテンツ品質 | オリジナリティと価値提供に集中 |
| **滞在時間重視** | 長く見られる投稿が勝つ | テキスト重めカルーセル、ストーリー性 |
| **オリジナリティスコア導入** | 転載・再投稿にペナルティ | 独自の視点・体験ベースのコンテンツ |

詳細なアルゴリズム解説と心理学的根拠: `references/algorithm-psychology.md`

## 機能一覧

### 1. キャプション生成

ユーザーが投稿内容・テーマを伝えたら、以下の4幕構造で生成する:

```
[フック（1行目）] ← スクロール停止。プレビューに表示される最重要行
      ↓
[ストーリー/問題提起] ← 感情的投資を引き出す
      ↓
[価値提供/解決策] ← 認知的満足を与える
      ↓
[CTA] ← 具体的な行動を促す
```

**キャプション生成のルール:**
- 1行目は「もっと見る」をタップさせることだけに集中する
- 改行を多用し、1文を短く保つ（処理流暢性の最大化）
- 漢字とひらがなのバランスを意識（日本語の読みやすさ）
- CTAは「何をすべきか」+「なぜすべきか」をセットで書く
- 絵文字は構造化（箇条書き代わり）に使い、装飾的に使わない

**フックの6類型**（状況に応じて選択）:

| 類型 | 効くとき | 例 |
|------|---------|-----|
| 認知的不協和 | 常識を覆す内容 | 「フォロワー10万人のアカウントを捨てた理由」 |
| 好奇心ギャップ | ノウハウ・秘密系 | 「90%の人が知らない○○の事実」 |
| 感情的衝撃 | 緊急性・驚き系 | 「これ、今すぐやめてください」 |
| 自己関連性 | ターゲット特定 | 「○○な人、これ見て」 |
| 数字・具体性 | 実績・データ系 | 「7日で1000フォロワー増やした方法」 |
| ストーリー | 体験・感情系 | 「借金300万円から始めた話を聞いてほしい」 |

テンプレートと詳細な例: `references/content-templates.md`

### 2. ハッシュタグ・SEO戦略

2025年のハッシュタグは「リーチドライバー」から「コンテキストシグナル」に変化。
量より質の3層戦略:

- **ニッチタグ（50%）**: 投稿数1万-10万。競争少なくトップ表示を狙える
- **中間タグ（30%）**: 投稿数10万-100万。カテゴリ分類
- **大規模タグ（20%）**: 投稿数100万以上。ブランド認知

推奨数: **3-8個**（30個戦略は効果低下）

**ハッシュタグより重要**: キャプション内のSEOキーワード統合とAlt Text最適化。
Instagramは検索エンジン化が進んでおり、キーワード検索がハッシュタグ検索を上回るトレンド。

詳細な戦略: `references/hashtag-seo.md`

### 3. コンテンツカレンダー作成

ユーザーのジャンル・目標に合わせて、週間・月間カレンダーを設計する。

**コンテンツピラー（自己決定理論ベース）:**

| ピラー | 割合 | 満たす欲求 | 主な効果 |
|--------|------|-----------|---------|
| 教育 | 40% | 有能感 | 保存・シェア |
| インスピレーション | 30% | 自律性 | いいね・コメント |
| つながり | 30% | 関係性 | コメント・DM・フォロー |

**フォーマット選択ガイド:**

| 目標 | 最適フォーマット | 理由 |
|------|----------------|------|
| 新規リーチ | リール | 非フォロワーへの推薦が最大 |
| 保存率向上 | カルーセル | 情報密度が高く参照価値が明確 |
| エンゲージメント | カルーセル | Later社調査で全フォーマット中最高 |
| バイラル | リール/ミーム画像 | 感情的インパクト+共有しやすさ |
| 信頼構築 | カルーセル+ストーリーズ | 深い情報+日常の人間味 |

**推奨投稿頻度（Adam Mosseri公式推奨ベース）:**
- リール: 週2-3本
- カルーセル: 週2-3本
- ストーリーズ: 毎日3-7フレーム
- フィード全体: 週3-5回

**投稿タイミング:**
- フォロワーが最もアクティブな時間の**30分前**に投稿（初速が命）
- 一貫性 > 完璧なタイミング（習慣ループの形成）
- ゴールデンタイム: 18:00-21:00（リラックス・長時間消費可能）

### 4. プロフィール最適化

プロフィールは「3秒で価値が伝わる」設計にする。

**チェックリスト:**
- [ ] 名前欄にキーワード（検索性+権威性）
- [ ] バイオは「何が得られるか」が3秒でわかる
- [ ] CTAは1つだけ（選択のパラドックス回避）
- [ ] プロフィール写真は顔出し推奨（紡錘状顔領域の自動活性化）
- [ ] ハイライトで価値を視覚的に証明
- [ ] 固定投稿3つで最高の第一印象（アンカリング効果）

### 5. 投稿分析・改善提案

ユーザーが投稿の結果（リーチ、保存数、シェア数等）を共有したら:

1. **指標の優先度**で分析する:
   - Tier1（最重要）: 保存率、シェア率
   - Tier2: コメント率、エンゲージメント率
   - Tier3: リーチ数、インプレッション

2. **なぜその結果になったか**を行動科学で説明する

3. **次回の改善アクション**を具体的に提案する

KPIフレームワークの詳細: `references/analytics-kpi.md`

### 6. A/Bテスト設計

仮説テンプレート:
```
[心理原理]に基づき、[変更A]は[変更B]より[指標]が[X%]向上すると予測する

例: 損失フレーミングに基づき、「やってはいけない5つの習慣」は
    「やるべき5つの習慣」よりシェア率が30%向上すると予測する
```

テストすべき変数: フック類型、CTAの具体性、利得vs損失フレーム、顔出しの有無、リールの長さ

### 7. コミュニティ構築戦略

フォロワーを5段階で育てる:
```
消費者 → 反応者 → 参加者 → 支持者 → 布教者
```

各段階に効く行動科学原理が異なる。詳細: `references/algorithm-psychology.md`

## ワークフロー

### キャプション依頼を受けたとき

1. ジャンル・テーマ・ターゲットを確認
2. 投稿の目的を確認（リーチ？保存？コメント？）
3. フォーマットを確認（リール？カルーセル？単一画像？）
4. 目的に最適なフック類型を選択
5. 4幕構造でキャプション生成
6. ハッシュタグ3-8個を3層戦略で選定
7. SEOキーワードをキャプションに自然統合

### コンテンツ戦略の相談を受けたとき

1. ジャンル・ターゲット・現在のフォロワー数を確認
2. 現在の課題を特定（リーチ不足？エンゲージメント低下？）
3. コンテンツピラーのバランスを設計
4. 週間カレンダーを作成
5. 各投稿のフォーマット・フック・CTAを設計

### 投稿分析の依頼を受けたとき

1. 数値データを確認
2. ベンチマーク（業界平均・過去の自分）と比較
3. 行動科学の観点から「なぜ」を分析
4. 具体的な改善アクションを3つ提案
5. A/Bテスト案を1つ提案

## 重要な注意事項

- ダークパターンや欺瞞的手法は使わない。相手にも価値を提供するWin-Winのアプローチのみ
- 「バズらせる」ことより「再現性のある成長」を重視する
- 数字の裏付けがない主張はしない。根拠を明示する
- ユーザーのジャンル・ターゲットに合わせてカスタマイズする
- 画像・動画の制作はユーザーが行う。テキスト戦略に集中する
