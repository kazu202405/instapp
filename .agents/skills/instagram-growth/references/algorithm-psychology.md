## 1. サーフェス別ランキングシグナル

### フィード

優先度順:
1. **関係性** - 過去のDM、コメント、タグ付け、プロフィール閲覧の履歴
   - 脳科学: 扁桃体+オキシトシン系が「親密な関係の人の情報」を優先処理
2. **関心度** - 過去に似たコンテンツに反応したか
   - 行動経済学: 確証バイアスで既存の関心テーマを優先取り込み
3. **適時性** - 投稿の新しさ + エンゲージメントの初速
   - 心理学: 新奇性バイアスでドーパミン系が「新しい情報」に強反応
4. **コンテンツ品質シグナル**
   - 滞在時間 > 保存数 > シェア数 > コメントの深さ（長文 > 絵文字のみ）

### リール

1. **視聴完了率** - 最重要。最後まで見られた割合 + リプレイ回数
   - BJ Foggモデルの「能力」: 短い動画ほど完了しやすい
2. **シェア数（Sends）** - 2025年最重要成長指標
   - DMでの共有がリーチ拡大の最大ドライバー
   - 社会的通貨理論: 「自分を賢く/面白く見せるコンテンツ」を共有
3. **保存数** - 「後で見返したい」意図的アクション = 高い重み
   - 保有効果の先行形: 保存時点で所有感が発生
4. **オリジナリティスコア** - 2024年導入
   - 再投稿・転載にペナルティ、オリジナルに「Original」ラベル
   - 10K以下のフォロワーにもリーチ機会拡大
5. **初速エンゲージメント** - 投稿後30-60分が勝負
   - バンドワゴン効果: 早期に盛り上がる投稿にさらに人が集まる

### ストーリーズ

1. 閲覧頻度 - そのアカウントのストーリーをどれだけ見ているか
2. インタラクション - リアクション、返信、スタンプ参加
3. 親密度 - DM・タグ付け頻度
4. ピーク・エンドの法則: 最後のフレームが全体印象を決定

### Explore

1. 類似ユーザー間での反応率（人気度）
2. Exploreでの過去行動履歴
3. コンテンツカテゴリの一致度
4. 投稿者の過去パフォーマンス

## 2. 行動科学の6大原理

### 2.1 BJ Foggの行動モデル（B = MAP）

**Behavior = Motivation x Ability x Prompt**

| 要素 | Instagram適用 | 具体施策 |
|------|-------------|---------|
| 動機を高める | 快楽/苦痛、希望/恐怖、承認/拒絶 | 感情的コンテンツ、損失フレーミング |
| 能力を高める（摩擦削減） | いいね=ダブルタップ、コメント=選択式 | 行動のハードルを下げる設計 |
| きっかけを設計 | CTA+視覚的キュー+タイミング | 「保存して後で見返して」+矢印 |

### 2.2 フックモデル（Nir Eyal）

**Trigger → Action → Variable Reward → Investment**

1. **トリガー**: 通知、ストーリーズのリング
2. **アクション**: スクロール、タップ
3. **変動報酬**: 予測不能なコンテンツ（スロットマシン効果）
   - 社会的報酬: いいね、コメント、フォロワー増
   - 狩猟報酬: 新情報、トレンド発見
   - 自己報酬: 達成感、スキル向上
4. **投資**: フォロー、保存、コメント（次の利用を確実に）

クリエイターとしての応用:
- 変動報酬の設計: コンテンツに予測不能性を持たせる（毎回同じはNG）
- 投資の促進: 保存コレクション構築、シリーズ化で次回期待
- 内部トリガー構築: 特定トピック→自分のアカウントを想起するポジショニング

### 2.3 チャルディーニの影響力6原則

| 原則 | Instagram適用 | 具体施策 |
|------|-------------|---------|
| 返報性 | 無料で価値提供 | テンプレート、チートシート公開 |
| コミットメントと一貫性 | 小さなYesから | アンケート→コメント→保存→フォロー |
| 社会的証明 | UGCとテスティモニアル | フォロワーの声、数字の提示 |
| 好意 | パーソナルブランディング | 顔出し、自己開示、ユーモア |
| 権威 | 専門性の証明 | 資格、実績、データ引用 |
| 希少性 | 限定コンテンツ | 期間限定、フォロワー限定 |

### 2.4 損失回避とプロスペクト理論

人は利得より損失に**約2倍**強く反応する（Kahneman & Tversky）。

- 「5つのコツ」（利得）< 「やってはいけない5つの間違い」（損失）
- ストーリーズの24時間消失 = 自然な希少性 + FOMO

### 2.5 認知負荷理論

ワーキングメモリは7±2チャンクしか処理できない（Miller, 1956）。

- 1投稿1メッセージの原則
- カルーセルは各スライドに1ポイント
- テキストは大きく、余白確保
- 色数3色以内

### 2.6 ストーリーテリングの神経科学（Paul Zak）

物語を聞いているとき:
- **コルチゾール**上昇 → 注意集中（「この先どうなる？」）
- **オキシトシン**分泌 → 共感と信頼形成
- **ドーパミン**放出 → 結末への期待（報酬予測）

キャプションでの応用:
1. 状況設定（共感できる場面）
2. 対立/困難（コルチゾールでテンション上昇）
3. 転換点（新しい気づき）
4. 解決/教訓（ドーパミン報酬）
5. 行動への招待（CTA）

## 3. エンゲージメント別の心理メカニズム

### フォローの心理学

自己決定理論（Deci & Ryan）の3欲求に対応:
- **有能感**: 「このアカウントで成長できる」→ 教育コンテンツ
- **自律性**: 強制ではなく価値提供後の自然な選択 → リアクタンス回避
- **関係性**: 「このクリエイターとつながりたい」→ パラソーシャル関係構築

フォローを促す認知バイアス:
| バイアス | 活用法 | 例 |
|---------|--------|-----|
| ハロー効果 | 第一印象最適化 | プロフ写真・バイオの質 |
| バンドワゴン効果 | 社会的証明 | フォロワー数、推薦 |
| 認知的不協和 | いいね→未フォローの矛盾 | 価値提供後のCTA |
| IKEA効果 | 参加型コンテンツ | アンケート、質問箱 |
| 希少性 | 限定感 | 「フォロワー限定で公開」 |

### いいねの心理学

最も認知負荷が低いエンゲージメント（ダブルタップで完了）:
- システム1の反応: 直感的・自動的
- ドーパミン報酬: いいねを「あげる」行為にも小さな報酬感（利他行動の神経基盤）
- 社会的シグナリング: アクティビティに表示 = 自分の価値観の発信

トリガー: 視覚的に美しい、感情的に共鳴、社会的に望ましい

### コメントの心理学

認知負荷が高い = より強い動機が必要:
- 自己表現欲求: 意見・経験・知識を表現したい
- 社会的承認: クリエイターからの返信期待
- 議論参加: 集団の一員として意見を述べたい（所属欲求）

トリガー: 明確な質問、安全な論争トピック、体験共有しやすい話題、返信文化

### シェアの心理学（2025年最重要）

Jonah Bergerの**STEPPS**モデル:
1. **Social Currency** - シェアで自分が賢く/面白く見える
2. **Triggers** - 日常で思い出される
3. **Emotion** - 強い感情（畏敬、驚き、笑い、怒り）
4. **Public** - 他の人がシェアしているのが見える
5. **Practical Value** - 役立つ情報を共有したい
6. **Stories** - 物語として伝えやすい

脳科学: 側頭頭頂接合部（TPJ）+内側前頭前皮質（mPFC）の活性化がシェア行動を予測（Falk et al., 2013）

### 保存の心理学

「未来の自分への投資」行為:
- 実装意図: 「後でこれを使おう」のif-then計画形成
- 損失回避: 「保存しなかったら情報を失う」
- 保有効果の先取り: 保存→「自分のコレクション」として所有感

保存されるコンテンツ: チートシート、チェックリスト、ステップガイド、繰り返し参照する情報

## 4. コミュニティ構築の5段階モデル

### 段階と行動科学原理

| 段階 | 目標 | 有効な原理 |
|------|------|-----------|
| 消費者→反応者 | いいね・保存・短いコメント | 変動報酬、低摩擦インタラクション、返報性 |
| 反応者→参加者 | 対話・ストーリーズ返信・UGC | IKEA効果、ラベリング、コミットメント |
| 参加者→支持者 | シェア・推薦・防衛 | 統一性原則、認知的不協和、エンダウメント効果 |
| 支持者→布教者 | 自発的宣伝・勧誘・共同創造 | 自己決定理論、社会的アイデンティティ、フランクリン効果 |

### 反復エンゲージメントの設計

**習慣ループ:**
- きっかけ: 投稿通知、一定の時間帯
- ルーチン: 投稿を見る、コメント、保存
- 報酬: 新知識、感情的満足、社会的つながり

**シリーズ化（ツァイガルニク効果）:**
- 「○○シリーズ Part 3/10」
- 毎週月曜の定番コーナー
- 月間チャレンジ

**予測可能性と変動のバランス:** 70%予測可能 + 30%新奇性

### DM戦略

DMが最強の関係構築ツールである理由:
1. プライベート空間 → 深い自己開示
2. 一対一の感覚 → パラソーシャルから「本当の関係」へ
3. 個人的返信 → 最強の返報性
4. 個人的会話 → 最もオキシトシンを分泌

施策:
- ストーリーズ反応者への個別返信
- 新規フォロワーへのウェルカムDM（手動 = 真正性）
- 質問箱回答をDMで送る（限定感 + 個人関係）

### 心理的安全性（Google Project Aristotle応用）

1. **安全性**: コメント欄でバカにされない雰囲気、失敗共有
2. **信頼性**: 予告したコンテンツを必ず投稿
3. **明瞭さ**: 参加方法の明示、コメントテンプレート
4. **意味**: 共通ミッションの共有
5. **影響力**: フォロワーのフィードバック反映

## 5. パターンインタラプトの科学

### なぜ人はスクロールし続けるのか

フィードスクロール = 習慣ループ（きっかけ:退屈→ルーチン:スクロール→報酬:新情報）
システム1（自動処理）の無意識的行動。

**止めるにはシステム2（意識的処理）を起動させる必要がある。**

### 神経科学的メカニズム

- 予測誤差（Prediction Error）→ ドーパミンニューロン発火
- 脳は「予測と異なるもの」に注意を向ける（生存のため）
- 新奇性は前頭前皮質と海馬を活性化

### リールの最初の1-3秒

- 0.5秒で「見るか見ないか」判断（システム1）
- 3秒で65%が離脱するか決定
- 構造: フック→コンテンツ→CTA

最初の1秒で使う要素:
1. **動き** - 方位選択性細胞の反応
2. **顔** - 紡錘状顔領域（FFA）自動活性化
3. **テキストオーバーレイ** - 二重符号化で記憶定着
4. **コントラスト** - 背景と被写体の明確な差異
5. **直接カメラ目線** - アイコンタクト効果
